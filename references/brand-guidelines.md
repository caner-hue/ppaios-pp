# Brand Guidelines — Purposeful Profits

Premium operator-led consultancy. Editorial, authoritative, warm. Not a startup, not an agency, not a coach.

## Colours

| Token | Hex | Role |
|---|---|---|
| `--ink` | `#0D0B09` | Primary background |
| `--ink-mid` | `#1C1915` | Card / surface |
| `--ink-light` | `#2A2520` | Elevated surface |
| `--purple` | `#BAA8E0` | Primary accent, CTA, eyebrows, rules |
| `--purple-dim` | `#8E7BC0` | Secondary accent |
| `--purple-deep` | `#4B3A7A` | Accent on light backgrounds |
| `--cream` | `#FFF6ED` | Light surface, primary text on dark |
| `--cream-dim` | `#E8DDD0` | Secondary text |
| `--cream-faint` | `#C8BAA8` | Muted text |
| `--grey-3` | `#9A8F84` | Placeholder, captions |
| `--grey-4` | `#5C5248` | Disabled |
| `--success` | `#4CAF7D` | Positive |
| `--error` | `#D94F3B` | Destructive |
| `--warning` | `#E8A838` | Caution |

Never use `#FFFFFF`. Cream for light, ink for dark.

## Typography
- Display / H1 / H2 / H3 / Eyebrow / Caption: **Syne**.
- Body / Italic: **Lora**.

Google Fonts: `family=Syne:wght@400;600;700;800&family=Lora:ital,wght@0,400;0,500;0,600;1,400;1,500`

## Spacing
4px base: 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96 / 128.

## Hard rules
1. No em dashes.
2. One primary CTA per view.
3. Purple is the only accent.
4. Syne for headings only. Lora for body only.
5. Cream not white for light surfaces.
6. Lead with proof: specific numbers, real brand names, documented outcomes.
7. Generous whitespace.
8. Eyebrows: purple, uppercase, Syne Bold.
9. Operator voice: direct, no therapy speak, no hype, sentences not bullets unless a list is genuinely the right shape.
